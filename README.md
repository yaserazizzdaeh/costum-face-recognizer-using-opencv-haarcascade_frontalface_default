# costum-face-recognizer-using-opencv-haarcascade_frontalface_default
costum face recognizer using opencv-haarcascade_frontalface_default
use datasamples.py to collecting your own dataset. if you can not i puted my own face dataset in dataset.zip
if you are interested to train your own model on your own dataset or on presented dataset use trainer.py, if you are not interested in just you presented pre-trained model that you can find it out in model.zip
at the end use recognizer.py to face recognizing for real-time
do not forget to have presented .xml file in your directory
